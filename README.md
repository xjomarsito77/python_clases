## comandos basicos terminal
- ````ls```` : listados de un archivo de una carpeta
- ````clear```` : limpiar una consola
- ````cd```` : navegar entre directorios
- ````rm```` : eliminar archivos
- ````rmdir````: eliminar carpeta
- ````mkdir```` : clear directorio
